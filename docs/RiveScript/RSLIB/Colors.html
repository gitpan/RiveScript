NAME

Colors.rsl - Library of Color Names

DESCRIPTION

This package exports the following arrays:

aColorName - An array of color names (even the obscure ones)

anEyeColor - An array of common eye colors

aWindowsColor - An array of the color names of the 16 standard Windows colors.

USAGE

  + my favorite color is (@aColorName)
  + my eyes are (@anEyeColor)
