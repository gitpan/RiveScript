NAME

RiveScript::Brain - Reply-getting routines for RiveScript.

DESCRIPTION

This module is used by RiveScript for handling the reply(), intReply(), and search() methods.

METHODS

reply ($RiveScript, $userID, $message[, %tags])

Returns a reply from the RiveScript brain. Takes the same arguments as RiveScript::reply()

intReply ($RiveScript, $userID, $message)

The internal reply-getting subroutine.

search ($RiveScript, $string)

Search the loaded replies for everything that matches $string.

SEE ALSO

The RiveScript manpage.
