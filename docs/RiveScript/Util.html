NAME

RiveScript::Util - Methods for the RiveScript brain.

DESCRIPTION

This module is used by RiveScript for performing common tasks.

METHODS

splitSentences ($splitters, $msg)

Take $msg and split it at $splitters, where $splitters is formatted like ``. ! ? ;''

formatMessage ($subs, $msg)

Format $msg using the substitutions defined in hashref $subs.

person ($subs, $text)

Run person substitutions in hashref $subs on data $text.

tagFilter ($RiveScript, $reply, $id, $msg)

Run tag filters on $reply using all the same arguments as RiveScript::tagFilter.

mergeWildcards ($string, $stars)

Merge wildcards from arrayref $stars into string $string.

stringUtil ($type, $string)

Run a text modifier on $string, where $type is one of uppercase, lowercase, formal, or sentence.
