NAME

RiveScript::Parser -- Read and Write routines for RiveScript documents.

DESCRIPTION

This module is used by RiveScript for reading and writing of RiveScript documents.

METHODS

new()

Create a new parser. Takes all the same arguments as RiveScript does.

debug()

Prints a debug message (internal).

loadFile()

Load in a single file. Takes the same arguments as RiveScript::loadFile

sortReplies()

Sort the replies. Takes the same arguments as RiveScript::sortReplies

sortThats()

Sort all the values of ``%PREVIOUS'' in the same fashion that replies are sorted.

write()

Writes the loaded replies to a file. Takes the same arguments as RiveScript::write

The RiveScript manpage.
